Git 個人的メモ

Jun 20, 2022

Memo, Git,

Git操作で役に立ちそうな個人的メモです

gitの一連の流れを自動化するbashファイル

以下のgit.shを.gitと同階層に置いた上で実行

git.sh

実行方法

コミットメッセージがUpdateで良い場合

$ bash git.sh

コミットメッセージを指定する場合

$ bash git.sh <commit message>

bashの省略(Mac)

ファイルに実行権限を与える

$ chmod +x git.sh

以後は以下のコメントのみで実行できる

$ ./git.sh

コミットメッセージを指定する場合は

$ ./git.sh <commit message>
