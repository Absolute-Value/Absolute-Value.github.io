## 操作方法 | Controls

自チームはD（青）です。表（D攻撃）はあなたが打撃・走塁を操作し、相手（C/赤）の投球は自動で行われます。裏（C攻撃）はあなたが投球を操作し、相手（C/赤）の打撃・走塁は自動で行われます。

### 操作方法（自チームの攻撃中） | Controls

| キー | 説明 |
| --- | --- |
| w / up | 上へ移動 |
| s / down | 下へ移動 |
| a / left | 左へ移動 |
| d / right | 右へ移動 |
| N（投球を受けている間） | 長押しでスイング、途中で離すことでバント |
| d / right | 一塁の走者を選択 |
| w / up | 二塁の走者を選択 |
| a / left | 三塁の走者を選択 |
| s / down | すべての走者を選択（打者が一塁へ走っている最中を除く） |
| 選択キー + M | 選択した走者を1つ先の塁へ進める |
| 選択キー + N | 選択した走者を、進塁を始めた塁まで戻す（それより後ろには戻らない） |

### 投球（自チームの守備中） | Pitching

ボールが死んでいて打者が打席にいる間（投球前）は、何もしなければピッチャーがランダムな時間（約1〜3秒）待った後に自動で投球します。以下のキーで、その前に自分から投球したり牽制したりできます。

| キー | 説明 |
| --- | --- |
| N | （待たずに）すぐ投球する |
| N + w / up | 遅い球を投げる |
| N + s / down | 速い球を投げる |
| （Nのみ） | 中くらいの速さの球を投げる |
| a / left, d / right | 投球中、ボールをカーブさせる（曲がりすぎないよう上限あり） |
| d / right + M | 牽制球を一塁へ投げる |
| w / up + M | 牽制球を二塁へ投げる |
| a / left + M | 牽制球を三塁へ投げる |

投球前は、ピッチャーはマウンドの白い棒の中だけ左右（a/d）に動けます。打たれた後は、ピッチャーを上下左右キーで自由に動かせます。

| キー | 説明 |
| --- | --- |
| a / left, d / right（投球前） | マウンドの白い棒の中でピッチャーを左右に動かす |
| w/a/s/d または矢印（打たれた後） | ピッチャーを自由に動かす |
| d / right + M（誰かがボールを持っている間） | 一塁へ送球する |
| w / up + M（誰かがボールを持っている間） | 二塁へ送球する |
| a / left + M（誰かがボールを持っている間） | 三塁へ送球する |
| s / down + M（誰かがボールを持っている間） | 本塁へ送球する |

ボールを拾った時点で誰もカバーしていない塁があれば、近くの野手が自動でその塁のカバーに入ります。

攻守交代すると、攻撃側・守備側チームの色が入れ替わります（表D=攻撃青/守備赤、裏C=攻撃赤/守備青）。
